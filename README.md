📌 Sobre la web

Este es un sitio de blogs reflexivos, con un tono más profundo y a veces triste, pensado para esos momentos en los que necesitás parar un poco y pensar.

Acá vas a encontrar textos sobre emociones, pensamientos, soledad, dudas y cosas que muchas veces no se dicen en voz alta, pero que todos sentimos alguna vez.

✍️ ¿Para qué es este blog?

La idea no es dar respuestas, sino acompañar.
Que puedas leer algo y decir: "no soy el único que piensa esto".
